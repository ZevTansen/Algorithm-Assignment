# Sorting Algorithms Assignment

## Description
This C++ program implements three sorting algorithms:
- Insertion Sort
- Merge Sort
- Quick Sort

It generates random datasets of sizes 10, 100, 1000, 10000, and 100000, runs each algorithm on them, and measures execution time in microseconds.
